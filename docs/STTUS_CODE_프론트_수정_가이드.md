# ARTAPPM STTUS_CODE 변경 — 프론트엔드 수정 가이드

백엔드에서 ARTAPPM(지원사업 신청) 테이블의 **STTUS_CODE** 코드 체계가 아래와 같이 변경되었습니다.  
프론트에서는 **코드 → 한글 라벨·필터 옵션·스타일** 매핑만 새 코드에 맞게 수정하면 됩니다.

---

## 변경된 코드 체계

| 코드 | 기존     | 변경 후        |
| ---- | -------- | -------------- |
| 01   | 임시저장 | 임시저장       |
| 02   | 신청     | 신청           |
| 03   | 승인     | 승인           |
| 04   | 반려     | **완료**       |
| 05   | 중단     | _(사용 안 함)_ |
| 11   | -        | **반려**       |
| 12   | -        | **중단**       |
| 99   | 취소     | 취소           |

- **한글 상태명**은 백엔드 API에서 이미 내려주므로, 프론트에서 **필터 옵션·배지/스타일용 코드 매핑**만 수정하면 됩니다.

## 3. 공통 참고

- **코드 값**: `"01"`, `"02"`, `"03"`, `"04"`, `"11"`, `"12"`, `"99"` (문자열)
- **한글**: 01 임시저장, 02 신청, 03 승인, **04 완료**, **11 반려**, **12 중단**, 99 취소
- **05는 더 이상 사용하지 않음** — 필터/옵션/switch-case 등에서 05 분기는 제거
